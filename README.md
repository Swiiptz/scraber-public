# Scraber

Scraber is an Android cyber watch app built with Flutter.

It tracks cybersecurity news, vulnerabilities, CVEs, vendor advisories and official alerts, with local favorites, calendar browsing and severity-based notifications.

## Status

Pilot release. Android first.

## Features

- Cybersecurity news feed
- CVE, vendor, product and tag search
- Severity levels: critical, high, medium, low
- Calendar view
- Local favorites
- Feed sorting by date or severity
- Native sharing for item titles and source links
- Push notifications by severity threshold
- Editorial UI inspired by cold paper, fine rules and issue numbering
